# 🚀 Scalable REST API with Authentication & Role-Based Access

## 📌 Overview
This project was built as part of the **Backend Developer Intern Assignment**.  
It demonstrates a secure, scalable backend system with **JWT authentication**, **role-based access control**, and **CRUD APIs**, along with a simple **React frontend** for testing and interaction.

---

## ⚙️ Tech Stack
### Backend
- **Node.js / Express.js**
- **MongoDB / Mongoose** (Database)
- **JWT Authentication**
- **BCrypt** for password hashing
- **Swagger** for API documentation

### Frontend
- **React.js (Vite)**
- **Axios** for API calls
- **React Router** for protected routes

---

## 🔑 Features
### Backend
- ✅ User Registration & Login (with hashed passwords & JWT)
- ✅ Role-Based Access (User vs Admin)
- ✅ CRUD APIs for `Tasks` entity
- ✅ API Versioning (`/api/v1/...`)
- ✅ Input Validation & Error Handling
- ✅ Swagger API Documentation
- ✅ Modular & Scalable Project Structure

### Frontend
- ✅ Register & Login Users
- ✅ JWT-Protected Dashboard
- ✅ Perform CRUD operations on `Tasks`
- ✅ Display Success/Error messages from API responses
