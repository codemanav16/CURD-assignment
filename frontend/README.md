# Frontend (Vite + React)

Lightweight authentication UI with login and sign up forms. The forms run fully on the client and include basic field validation and demo success/error messages.

## Available scripts

- `npm run dev` – start the Vite dev server.
- `npm run build` – build for production.
- `npm run preview` – preview the production build.
- `npm run lint` – run ESLint.

## Notes

- Backend calls are wired to `VITE_API_BASE` (defaults to `http://localhost:3001`).
- Styles live in `src/App.css`; layout resets in `src/index.css`.
